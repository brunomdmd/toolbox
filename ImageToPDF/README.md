# ImageToPDF

Esse script surgiu de uma situação em que eu precisava enviar alguns documentos e comprovantes por meio de um formulário, tirei todas as fotos e só no momento em que fui fazer o upload das imagens, o formulário dizia que só aceitava PDF rsrs, pensei: "não vou fazer um a um de novo"... aí saiu esse script.

## Como usar

```bash
git clone https://github.com/your-username/toolbox.git
cd toolbox
python ImageToPDF/image_to_pdf.py
```
